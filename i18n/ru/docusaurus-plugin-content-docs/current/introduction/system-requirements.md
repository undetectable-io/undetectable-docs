## Minimum system requirements

- 4 ГБ ОЗУ;
- 1 ГБ свободного места на жестком диске.

**Поддерживаются только 64-битные операционные системы.**

Поддерживаются MacOS с процессором M1.

Браузер тестировался на:

- Windows 7 и выше;
- MacOS High Sierra;
- MacOS Catalina.

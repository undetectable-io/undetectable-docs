# System Requirements

## Minimum system requirements
- 4GB RAM
- 1 GB of free hard drive space

Supported operating systems
Only 64-bit operating systems are supported

MacOS with the M1 processor is supported

The browser was tested on:

- Windows 7 and later
- MacOS High Sierra
- MacOS Catalina
